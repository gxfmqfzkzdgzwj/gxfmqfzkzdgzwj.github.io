github常用指令

2017-11-30

首先，当你完成git安装后，将github的用户名和邮箱进行配置

如何使用github提交代码

  1. 用github建仓库
  2. 进入目录(或者直接在目录下右键git bash here)
  3. git init
  4. git status
  5. git add .
  6. git status
  7. git commit -m ‘注释’
  8. git push -u 网址 master
    注意：网址是你建立仓库时对应的网址,以http开头，.git结尾

如何预览github里的demo
. 进入到你想要上传的项目文件夹下
. git init
. 创建gh-pages 分支， git checkout –orphan gh-pages
. 添加文件到暂存区，git add .
. 添加信息 git commit -m ‘This is add message’
. 添加仓库 git remote add origin git@github.com:username/project.git
. 部署你的项目到github，git push origin gh-pages
. 接下来你在当前仓库下的.io就可以看到你的项目Demo了
