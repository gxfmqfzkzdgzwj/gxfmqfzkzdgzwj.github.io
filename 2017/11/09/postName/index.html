HEXO 个人博客搭建

2017-11-09

HEXO介绍

Hexo 是一款基于Node.js、快速、简洁且高效的博客框架。
Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。详情请参考https://hexo.io/zh-cn/docs/index.html

安装
安装hexo特别简单。在安装之前确保你的电脑上已经安装以下应用程序

  • Node.js
  • Git

如果你的电脑上已经安装了上述程序，接下来直接使用npm即可完成安装。
npm install -g hexo-cli

  2. hexo博客的next主题在github上显示不出页面内容，只有框架，别的主题正常，本地显示也正常。
    解决办法：
hexo markdown
