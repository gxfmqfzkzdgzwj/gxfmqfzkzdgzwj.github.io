CSS的预处理框架

Less

  1. 安装node.js
  2. 打开命令行
  3. npm install -g less 安装less
  4. 进入目录 cd
  5. lessc styles.less
  6. lessc styles.less styles.css
    这样就可以将less文件 转化为CSS文件

Stylus

安装

  1. 安装node.js
  2. npm install stylus -g
  3. 生成CSS
    建立一个stylusExample/，再在里面建立 src 目录专门存放 stylus 文件，在里面建立 example.styl 文件。然后在 stylusExample 目录下面执行下面命令
    $ stylus --compress src/

输出compiled src/example.css ，这个时候表示你生成成功了，带上–compress参数表示你生成压缩的CSS文件。

leetcode总结

第二个 Add Two Numbers

分情况判断

  1. 两个节点中只要有一个为空，返回节点为另一个节点
  2. 采用构造函数，构造一个新节点，传入值为 两节点值的和
  3. 下一个节点 等于两节点下一个节点的和
  4. 对当前节点进行判断，如果当前节点值大于9，则令当前节点值减去10
  5. 对于当前节点的下一节点，值在原来的基础上加一。相当于addTwoNumbers(l.next,new ListNode(1))。即将此时当前节点的下一节点与值为1的节点相加。
  6. 不断循环，不断嵌套，即可得到最后的节点。

第三个 Longest Substring Without Repeating Characters

算法思想：
第一种：

  • 对所有的字符串进行枚举
  • 然后对所有枚举后的字符串进行判断 是否重复
  • 这种算法时间复杂度太高

第二种

  • 截取字符串（i，j）
  • 判断第j个字符是否在截取的字符串内
  • 如果存在
  • 让i=所在位置+1
  • 继续截取
  • 不断重复
  • 最后len= Math.max(len,j-i+1);

一.display:none和 visibility：hidden的区别

二.首先我们要明白的是，页面的显示过程分为以下几个阶段：

  1. 生成DOM树(包括display:none的节点)

  2. 在DOM树的基础上根据节点的集合属性(margin,padding,width,height等)生成render树(不包括display:none，head节点，但是包括visibility:hidden的节点)

  3. 在render树的基础上继续渲染颜色背景色等样式

reflow:当render树的一部分或者全部因为大小边距等问题发生改变而需要重建的过程，叫做回流

repaint:当诸如颜色背景等不会引起页面布局变化，而只需要重新渲染的过程叫做重绘

四.History对象的属性和方法

四.只有实例对象上不存在的属性和方法才会去原型上查找。

prototype 原型。。
构造原型(construtor prototypes)

什么是原型？

  • 原型是一个对象，其他对象可以通过它实现属性继承。

任何一个对象都可以成为原型么？

  • 是

哪些对象有原型？

  • 所有的对象在默认的情况下都有一个原型，因为原型本身也是对象，所以每个原型自身又有一个原型.(只有一种例外，默认的对象原型在原型链的顶端。)

  • 在javascript中，一个对象就是任何无序键值对的集合,如果它不是一个主数据类型(undefined，null，boolean，number，or string)，那它就是一个对象。

  • 因为在javascript中，直接通过function申明的函数，后面定义的，会影响到之前的引用。
    通过var申明的函数，后面定义的并不会影响前面的引用。

  • 通过function定义的函数，后面定义的，照旧影响了前面的引用，但是不能改变通过var申明函数后的引用。

  • 反而，通过var申明的函数，改变了后来通过function申明函数之后的引用。

五.absolute和relative的区别。

定位为relative的元素脱离正常的文本流中，但其在文本流中的位置依然存在，
而定位为absolute的层脱离正常文本流，但与relative的区别是其在正常流中的位置不在存在。
