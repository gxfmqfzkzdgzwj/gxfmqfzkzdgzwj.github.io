  1. 在CSS2.1中，&不是一个有效字符
    CSS基本语法以及可能出现的不合法表示
    代码如下：
    h3, h4 & h5 {color: red }
    因为&不合法，所以CSS会忽略此行代码

  1. 百分比
    百分比一般与另一个属性相关联。每个可以用百分比表示的属性一般都定义了是哪个其他属性的百分比。
    p { font-size: 10px }
    p { line-height: 120% } / 120% of ‘font-size’ 12px/

基本掌握，但还有一点需要深入的知识点如下：

长度单元有两种形式：相对型和绝对型。相对长度单位指定相对于另一个长度属性的长度，例如ex和em
em:与字体的font-size相关
ex：以给定字体的小写字母x的高度为基准
