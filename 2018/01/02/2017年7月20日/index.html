三个 CSS 预处理器（框架）：Sass、LESS 和 Stylus

  • 语法
    首先 Sass 和 Less 都使用的是标准的 CSS 语法，因此如果你可以很方便的将已有的 CSS 代码转为预处理器代码，默认 Sass 使用 .sass 扩展名，而 Less 使用 .less 扩展名。 Stylus 支持的语法要更多样性一点，它默认使用 .styl 的文件扩展名，我们使用缩排和凹排代替花括号。

引入方式：

  • 变量

Stylus 对变量名没有任何限定，你可以是 $ 开始，也可以是任意的字符，而且与变量值之间可以用冒号、空格隔开，需要注意的是 Stylus (0.22.4) 将会编译 @ 开始的变量，但其对应的值并不会赋予该变量，换句话说，在 Stylus 的变量名不要用 @ 开头。
