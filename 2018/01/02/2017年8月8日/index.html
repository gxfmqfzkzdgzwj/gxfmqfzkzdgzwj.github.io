Leetcode 刷题总结

Posted on 2018-01-02

2017年8月8日

leetcode刷题总结

  1. Big Countries

  2. Merge Two Binary Trees（合并两个数结构）
    复习过程中要结合ife task22 任务地址
    二叉树的遍历方法。
    中间过程中采用迭代方法。层层迭代，像洋葱一样一层一层剥开我的心。

  3. Judge Route Circle
    解决问题的关键是设出该点的坐标，对于不同的变换，通过switch进行判断，看最后坐标是否在远点。

  4. Array Partition I
    个人理解是先利用sort方法对数组进行排序，当从小到大进行排序时，将第奇数个数相加，所求即为所得。

  5. Swap Salary
    利用UPDATE来进行更新
    UPDATE salary set sex = CHAR(ASCII(‘f’)^ ASCII(‘m’)^ ASCII(sex))

  6. Number Complement
    主要用到的知识点：
    parseInt(“11”，2);等于3；parseInt可以将一个数字字符串转换为对应的进制数。
    split函数可以将一个字符串分成一个字符数组。join()方法可以将字符数组变成一个字符串
    tostring()方法可以把数值型转换成相应的进制。例如3变成011.最后变成了一个字符串。
